- [telnet 与 ping 命令的区别]()

telnet 语法：telnet ip port

例子: telnet 34.64.20.32 18001

ping 语法: ping ip

例子：ping 34.64.20.32 或者 ping http://www.baidu.com

总结:

  1. telnet 是检测某台机器的某个程序能否访问
  2. ping 是检测某台机器能否访问 ```
